Rangoon VPN

Bypass internet censorship in Myanmar with secure and private connections.

Powered by Xray, Hiddify-Manager, v2ray-core

Why?

Traditional VPNs and common protocols like IKEv2, WireGuard, and OpenVPN are ineffective due to Myanmar military regime's recent use of advanced traffic analysis systems similar to China's Great Firewall.

Access to Rangoon VPN will only be manually granted temporarily to prevent abuse and manage server costs.

For access, please contact on Telegram: @rangoonvpn

This project is run by a college student for free, and keeping the servers up can get pretty pricey(really pricey).

If you find this project useful or impactful, please consider donating to help keep the service running.
